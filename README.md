# UI Kit Project

## Overview

This project is a **UI Kit** designed to provide a structured and customizable design system for web applications. It includes a set of predefined colors, typography, and components to ensure consistency across your project.

## Features

- **Colors**: Primary and secondary colors with state indicators.
- **Typography**: Predefined font styles for consistency.
- **Iconography**: Standardized icons for UI elements.
- **Grid System**: A flexible layout grid for responsive design.
- **Spacing**: Defined spacing units for layout alignment.
- **Textfields**: Styled input fields for forms.
- **Selectors**: Custom dropdowns, checkboxes, and radio buttons.
- **Buttons**: Pre-styled buttons with different states.
- **Small Elements**: Badges, chips, and labels.
- **Big Elements**: Modals, popups, and large UI components.
- **Other Elements**: Utility components for UI consistency.
- **Cards**: Predefined card styles for content grouping.

## Installation

To use this UI Kit in your project, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/amandafcaramico/UIKit.git
   ```
2. Navigate to the project directory:
   ```sh
   cd ui-kit
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
